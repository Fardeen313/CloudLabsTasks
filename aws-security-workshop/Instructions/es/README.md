# Taller Avanzado de Seguridad en AWS

Bienvenido al taller avanzado de seguridad en AWS.

En este laboratorio aprenderá prácticas avanzadas de seguridad en AWS.

## Lo que aprenderá

- AWS IAM
- Monitoreo con CloudTrail
- Buenas prácticas de seguridad

Cada participante recibirá una cuenta AWS dedicada y una máquina virtual.
