# Exercise 3 – CloudTrail Monitoring

AWS CloudTrail records activity across your AWS account.

## Task 1: Open CloudTrail

1. In the AWS Console search for **CloudTrail**.
2. Click the CloudTrail service.

## Task 2: Create Trail

1. Click **Create trail**.
2. Trail name:

SecurityTrail-<inject key="AWSAccountID"/>

3. Enable **All Regions**.
4. Click **Create Trail**.

## Wait Time

Trail creation may take **2-3 minutes**.

## Task 3: View Logs

1. Open **Event History**.
2. Observe recorded AWS API actions.

## Summary

CloudTrail is now enabled for monitoring your AWS account.
