# Exercise 1 – Getting Started

In this exercise, you will access your AWS account and verify your environment.

---

| Field | Value |

## Task 1: Open AWS Console

### Steps

1. Open **Brave Browser** from the virtual machine desktop.

2. Navigate to the AWS Console URL:
```
   https://console.aws.amazon.com
```

3. Sign in using the credentials provided in the **Environment Details** tab on the left panel.

4. Once logged in, verify your AWS Account ID matches the value below:

   > ✅ **Expected Account ID:** <inject key="AWSAccountID" enableCopy="true" style="color:blue"/>

---

> **Note:** If your Account ID does not match, please reach out to your instructor before proceeding.
